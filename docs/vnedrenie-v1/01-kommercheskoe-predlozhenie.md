## 1. Предмет договора

Разработка (доработка при необходимости), поставка, развёртывание на площадке Заказчика и ввод в эксплуатацию программного комплекса для:

- загрузки и разбора данных из источников: управляющая компания (УК), брокер, депозитарий (ДЕПО);
- описания задачи пользователем на естественном языке и интеллектуальной настройки парсинга (генерация правил) при нестандартных форматах выгрузок УК;
- выполнения сверки (аудита) по согласованным правилам между источниками;
- просмотра результатов, фильтрации и выгрузки отчётов (в т.ч. в CSV).

Все компоненты, включая **языковую модель**, размещаются **в инфраструктуре Заказчика**; обработка данных производится **без передачи в сторонние облачные сервисы** (после внедрения), за исключением случаев, отдельно оговорённых сторонами в договоре.

## 2. Состав поставки (версия 1)


| №   | Элемент         | Описание                                                                                                                         |
| --- | --------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| 1   | Веб-приложение  | Интерфейс загрузки файлов, просмотра данных, запуска аудита, экспорта                                                            |
| 2   | Серверная часть | API, парсинг Excel и PDF, сохранение в базу данных                                                                               |
| 3   | База данных     | PostgreSQL, схема под хранение сделок по источникам                                                                              |
| 4   | Модуль ИИ       | Локальный сервис LLM + интеграция с приложением для генерации JSON-правил парсинга по превью файла и тексту запроса пользователя |
| 5   | Документация    | Инструкция администратору (развёртывание, резервное копирование), краткая инструкция пользователю                                |
| 6   | Обучение        | Согласуется с Заказчиком (ориентир: 1–2 занятия с ключевыми пользователями)                                                      |


Исключения (не входят в v1 без отдельного соглашения): интеграция с 1С/учётными системами Заказчика, единый вход через Active Directory, круглосуточная поддержка с SLA, доработка под новые типы отчётов сверх согласованного перечня форматов.

- Начало работ: после поступления аванса и предоставления Заказчиком доступа к стенду/железу и образцам форматов (при необходимости — под NDA).
