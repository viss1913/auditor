Расшифровка ОС и кап. вложений
Имеется выгрузка ОСВ по сч. 01 и/или ведомость по амортизации с субконто и категоризацией, отличающимися в зависимости от базы 1С, её наполнения. В примере выгрузка по сч. 01 представлена в виде ведомости по амортизации. 
•	Выгрузка по сч. 08 в примере представлена в виде ОСВ (для демонстрации разницы в виде выгрузок). Для целей различных расшифровок по различным счетам могут использоваться разные значения из ОСВ – обороты по Дт и/или Кт, остатки по Дт и/или Кт, сальдо. 
Для целей автоматизации возможна выгрузка с максимальным количеством субконто (при условии возможности обработки). 
Выгрузка может быть представлена в виде неструктурированных таблиц с несколькими под итогами (иногда в одном стиле, иногда в разных стилях, с автогруппировкой или без неё).
Задачей минимум любой автоматизации является формирование «плоской» таблицы (по сути таблицы в формате, читаемой для любой БД), для целей дальнейшей обработки Pivot и/или мэппинга. Дополнительная обработка, анализ, категоризация и т.д. не предполагается. 
Задачей максимум дополнительно является:
•	непосредственное формирование pivot (или иной таблицы с возможностью выбора данных, фильтрации и др.)
•	получение данных, содержащихся в одной ячейке и создание дополнительных значений (например, одна ячейка может содержать текстовые данные «название ОС, артикул, серийный номер, дата постановки на учёт». Может быть потребность выделить из указанного значения, например, только дату или серийный номер. Формат может отличаться для разных объектов / объектов, добавленных в разные годы и т.д. 
•	результирующих таблиц для отчёта (без технических данных, в формате Мариллион)
•	автопроверка всех значений, в том числе итого и подитогов
